# CodeAlpha_FAQChatbot
AI FAQ Chatbot using Python 
## Project Description
This project is an AI FAQ Chatbot that answers user questions using Natural Language Processing (NLP) techniques.

## Features
- FAQ Question Answering
- Interactive Chat Interface
- AI Knowledge Base
- User-Friendly Design
- Suggested Questions

## Technologies Used
- HTML
- CSS
- JavaScript
- NLP Concepts
- Cosine Similarity

## Sample Questions
- What is AI?
- What is Machine Learning?
- Explain ChatGPT
- What is NLP?
- What is Computer Vision?

## How It Works
1. User enters a question.
2. The chatbot processes the text.
3. The system matches the question with the closest FAQ.
4. The chatbot displays the best answer.
